11 Delegates and Class Delegates

11.1 Delegates

11.1.1 Syntax

delegate → delegate‑specifiers delegate return‑type delegate‑name parameter‑list ;
delegate‑specifiers → access?
delegate‑name → identifier

11.2 Class Delegates

11.2.1 Syntax

class‑delegate → class‑delegate‑specifiers class delegate return‑type class‑delegate‑name parameter‑list ;
class‑delegate‑specifiers → access?
class‑delegate‑name → identifier
