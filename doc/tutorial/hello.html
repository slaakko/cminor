3.3 Inspecting the Contents of an Assembly

To inspect the contents of an assembly file, we use the cminor dump command. The dump contains virtual machine instructions for each compiled function. After that symbols of the hierarchical symbol table are printed.

> cminor dump assembly/debug/hello.cminora hello.dump
> type hello.dump
main() [internal System.Void main()]
{
0 BC receive
1 C3 enterblock
2 B9 loadconstant.b 128
3 DE slit2s
4 C0 call System.Console.WriteLine(System.String)
5 C4 exitblock
